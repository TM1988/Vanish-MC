# Vanish

Small Fabric mod that adds a toggleable trash slot to the player inventory.

## Versions

- Minecraft 26.1
- Fabric Loader 0.18.5
- Fabric API 0.144.3+26.1

## What It Does

- Press T while inventory is open to show or hide the trash slot.
- Press Delete on a hovered inventory item to move it into trash.
- Press Delete while hovering the trash slot to clear it.

## Build

On Windows:

```bat
gradlew.bat build
```

Output jar is created in build/libs.

## Test In Dev Client

```bat
gradlew.bat runClient
```
